SharpSvn is a binding of the Subversion Client API for .Net 2.0 applications contained within a set of xcopy-deployable dll's. The primary users of this api are (at this time):
  • AnkhSVN 2.0 - Subversion integration for Visual Studio 2005 and 2008
  • StudioTurtle - Subversion integration in Visual Studio 2005+ (Prototype stage; mostly integrated in AnkhSVN 2.0)
  • The SourceServerSharp Project
  • The TurtleBuild Project
