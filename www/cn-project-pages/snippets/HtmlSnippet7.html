What's New


SharpSvn is a binding of the Subversion Client API for .Net 2.0 applications. All this contained within a set of xcopy-deployable dll's.The primary users of this api are (at this time):
  • Ankhsvn 2.0 - Subversion integration for Visual Studio 2005 and 2008
  • StudioTurtle - Subversion integration in Visual Studio 2005+
  • The SourceServerSharp Project
  • the TurtleBuild Project
The API encapsulates the whole Subversion 1.5 client api and a few of the more low-level api's. The api is designed around the Microsoft class library design guidelines and follows the common language specification (CLS). The SharpSvn managed code includes all subversion references, so you can just use xcopy deployment for releasing applications using this libary.

The implementation is mostly complete and a test project has been added which tests most of the api. As the Subversion 1.5 api has not been declared stable the sharpsvn api must be treated as unstable (the api can change until the initial 1.5 release of SharpSvn).

We are seeking for help building a more complete test set and are open for extending the api for more use-cases.
