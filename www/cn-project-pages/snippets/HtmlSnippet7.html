What's New


SharpSvn is a binding of the Subversion Client API for .Net 2.0 applications. All this contained within a set of xcopy-deployable dll's.The primary users of this api are (at this time):
  • Ankhsvn 2.0 - Subversion integration for Visual Studio 2005 and 2008
  • StudioTurtle - Subversion integration in Visual Studio 2005+
  • The SourceServerSharp Project
  • the TurtleBuild Project
